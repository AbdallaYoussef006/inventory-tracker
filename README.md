# 📦 Inventory Tracker

A **desktop inventory management application** built with C++/CLI Windows Forms in Visual Studio — featuring user authentication, product management, customer tracking, and order processing.

## ✨ Features

- 🔐 **Login System** — Secure user authentication with file-based storage
- 📝 **Sign Up** — New user registration
- 🏠 **Home Dashboard** — Overview and navigation hub
- 📦 **Products Management** — Add, view, and manage inventory items
- 👥 **Customer Management** — Track customer records
- 🛒 **Orders Management** — Process and manage orders

---

## 🗂️ Project Structure

```
inventory-tracker/
├── main.cpp           # Application entry point
├── Login.h / .cpp     # Login form & authentication logic
├── SignUp.h / .cpp    # Registration form
├── Home.h / .cpp      # Main dashboard
├── Products.h / .cpp  # Product management screen
├── custmors.h / .cpp  # Customer management screen
├── ordersform.h / .cpp # Orders management screen
├── users.txt          # User credentials storage
└── Login.txt          # Login data file
```

## 🚀 How to Run

**Requirements:** Visual Studio 2022 with C++/CLI support (Windows only)

1. Open `LoginForm.sln` in **Visual Studio**
2. Set build configuration to **Debug x64**
3. Press **F5** to build and run
